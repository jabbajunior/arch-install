# Arch Linux Installation Project

This repository contains the documentation for my personal Arch Linux installation project. It includes the planning, architectural reasoning, installation process, key learnings, and a day-by-day log of progress.

## Purpose

The goal of this project was to complete an Arch Linux installation while developing a stronger understanding of Linux fundamentals, system configuration, and post-installation setup. In addition to completing the install itself, I documented the process to capture my reasoning, reinforce learning, and create a reference for future work.

## Documentation Structure

### 01 Goals
Outlines the purpose of the project, the learning objectives, and the scope defined at the beginning.

### 02 Architecture
Explains the architectural and design decisions made during the project, along with the reasoning behind them and how they supported the project scope.

### 03 Installation
Documents the commands used and the steps taken to perform the Arch Linux installation.

### 04 Learnings
Expands on topics that were especially valuable during the project. This section also includes post-installation tasks such as working with `systemd` and configuring the system firewall.

### 05 Log
Provides a detailed daily log describing what I worked on throughout the project.

## Summary

This project was built as both a hands-on learning exercise and a record of the full installation process. Rather than documenting only the final result, this repository also captures the decisions, lessons, and progress made along the way.
